ABOUT THE API

This REST API provides the content of a given iCal file in a JSON format. This makes it easy to use iCal files for (web) applications. Keep in mind that the iCal file you want to use needs to be hosted somewhere on the internet, or else we can't find it. :(

CUSTOMIZE THE API

In this section you can enable/disable testing mode and change a personal time offset. The testing mode changes the output of the API so that it always returns a test event that is set to happen 2 minutes after the API call. This can be useful for testing the API with our own IoT alarm! The personal time offset allows you to provide a time offset to the output JSON. This can be used to make our IoT alarm make some noise in a timely manner, as an alarm that rings when the event has just started is what we call 'useless'.
