iCal
REST API
ABOUT THE API

This REST API provides the content of a given iCal file in a JSON format. This makes it easy to use iCal files for (web) applications. Keep in mind that the iCal file you want to use needs to be hosted somewhere on the internet, or else we can't find it. :(

Request URL

Necessary HTTP Headers

Key: "calendarurl" & Value: Your calendar url
